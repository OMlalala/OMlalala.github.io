总结¶

根据 2018年总结 中的承诺, 今天补交2019年第1周总结.

蟒营¶

2018年的尾巴, 大妈的 蟒营 Python 入门班 开班了.

入营目的¶

公开渠道推荐后, 有四个朋友咨询, 沟通后, 发现有一个好朋友最合适, 就推荐入营了.

因为我之前已经参加过大妈的课程: 开智学堂编程思维 Python 基础班三期, 所以这次入营有两个目的

  • 协助好朋友度过课程中的 “至暗时刻”, 增加通关概率
  • 再次培养/提升自己的编程能力
    • 持续 logging 的基本功. (向 @小焱 学习)
    • 输出习惯

意外收获¶

发现还有一个老乡 @recaff 也在嗯哼, 才发现还在其他地方相识过…(开智写作五期, 自怼圈)

希望能彼此鼓励, 交出一个满意的大作业.

今年的两个关键字是 “聚焦” 和 “创造”.

本周在聚焦方面做的好的:

  • 手机上卸载了 知识星球

    • 之前的查看频率是每周会在手机端查看多次.

本周在聚焦方面做得不好的:

  • 没有按时嗯哼 CFA 复习, 输出笔记.
